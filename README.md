# Employee Management System

## Project Overview
This is a Spring Boot REST API project for managing employees.

The project performs CRUD operations:
- Create Employee
- Read Employee
- Update Employee
- Delete Employee

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman
- GitHub

## REST APIs

### GET Employees
GET /employees

### Add Employee
POST /employees

### Update Employee
PUT /employees

### Delete Employee
DELETE /employees/{id}

## Database
MySQL database is connected using Spring Data JPA.

## Testing
APIs tested using Postman.
